# Profac

Profac is a application that we (me and [Aiman](https://github.com/PlashSpeed-Aiman)) build for Hackathon GDTA 2023. The hackathon theme was something like "Map" if I remember correctly. So we decided to build a application that can help people to find the nearest public facilities in the kulliyyah (faculty) in our university.

The iteration is very basic and has a lot of rough edges, but we learned a lot from this project nevertheless.

The floor plan was obtained from https://github.com/KOE-Wayfind/koe-floor-plans. The floor plan was traced in Figma and exported to SVG format, which this app uses.

## Getting Started

The app can be ran on any platform that Flutter supports. Run the following command to run the app:

```shell
flutter pub get
flutter run
```
